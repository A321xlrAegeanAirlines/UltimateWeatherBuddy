A desktop weather app built with Tkinter + Open-Meteo.

Features

  • 12-day forecast
  • Hourly graphs + comfort index
  • Favorites + compare window
  • Light/Dark themes, Metric/Imperial

Download / Run

Clone the repo or download the code:

pip install -r requirements.txt
python weather_buddy.py
